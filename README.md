# Pytex

## Запуск

Скопировать `.env.example` в `.env.prod`.

Отредактировать `.env.prod` и `docker-compose.yml` под свои параметры портов и адресов.

Запустить  `make build`

Запустить `docker compose up`

## Тестирование

Запустить `make test`

## Покрытие тестами

Запустить  `make test-cov`

## Возможные проблемы

Иногда на widows системах pydantic_settings не перечитывает измененные данные `.env`, для устранения необходимо очистить байт код проекта.

Запустить `make clean`
